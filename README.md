# Three.js template

[Live demo]()

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies
npm install

# Serve at localhost:3000
npm run start

# Build for production in the dist/ directory
npm run build
```

## Overview
Template consists of:
- Three.js
- Blacksmith
- tweakpane
